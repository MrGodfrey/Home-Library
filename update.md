## 更新

给我加一个功能，就是书籍的封面图片。我可以从网上下载图片，然后在网站上自己添加。

具体要求如下：
1. 开发一个上传图片的功能
2. 图片应当存储在 Cloudflare 的 R2 存储桶之中
3. 把相关的配置指南写到 README 之中

如果说它有本地图片的话，你就得优先使用 R2 存储桶中的图片。

然后将现在已经有的书籍封面连接 URL 取消掉，把它改成上传图片的功能。
